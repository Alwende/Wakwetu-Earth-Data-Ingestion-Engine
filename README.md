# Wakwetu Earth-Data Ingestion Engine 🌍

## The Sovereign Intelligence Bridge
A production-grade geospatial lakehouse engineered to ingest, secure, and elicit intelligence from high-resolution satellite imagery (Sentinel-2).

### 🚀 Live Intelligence Dashboard
[View the Live Portal](https://Alwende.github.io/Wakwetu-Earth-Data-Ingestion-Engine/)

### 🧠 Core Architectural Pillars
1. **Sovereign Ingestion:** Managed 207.9 MiB multipart streaming (Boto3) from us-west-2 to us-east-1.
2. **Analytical Elicitation:** AI-driven feature extraction (Urban Density & Anomaly Detection).
3. **Serverless Discovery:** SQL-based metadata querying via Amazon Athena and AWS Glue.
4. **Automated Governance:** 100% IaC (Terraform) and CI/CD (GitHub Actions).

### 🛠 Tech Stack
- **Cloud:** AWS (S3, Athena, Glue, Lambda)
- **Engine:** Python / Boto3
- **Frontend:** React / Tailwind
- **DevOps:** Terraform / GitHub Actions
